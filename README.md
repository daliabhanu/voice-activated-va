# Voice Assistant Project

A Python-based voice-activated assistant with a simple interface and local storage.

## Features
- Voice-to-text and text-to-speech
- Reminders stored locally in `assistant_data.json`
- Basic commands for time, reminders, and responses

2. Install dependencies:

   pip install SpeechRecognition pyttsx3 tkinter json transformers
